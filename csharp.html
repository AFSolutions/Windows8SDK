PayPal Checkout C# API

The PayPal namespace currently contains the following objects:

  • PayPal.Checkout.BuyNow - The PayPal Checkout API
  • PayPal.Checkout.Item - A model object used to describe an item being purchased
  • PayPal.Checkout.ItemBuilder - The builder used to create an Item

PayPal.Checkout.BuyNow object

This is the main PayPal checkout API. You can code against this if you have your own UI control or other mechanism to trigger purchases.

Constructor
BuyNow(String merchantId) Creates an instance of the BuyNow object for the provided merchantId (the emailid of your PayPal Merchant account).
Events
Attach event handlers on the BuyNow object to be notified of important events. For example, {% highlight csharp %} PayPal.Checkout.BuyNow bn = new PayPal.Checkout.BuyNow(); bn.Start += new EventHandler((source, args) => { // Do something });{% endhighlight %}
Event Description event
onstart Occurs when a transaction is initiated. Start
onauth Occurs when buyer authorization is triggered. Auth: token
oncancel Occurs when a user cancels the transaction.
Note: complete and error will not fire.		 Cancel: token
oncomplete Occurs when a user successfully completes a transaction.
Note: cancel and error will not fire.		 Complete: token, transactionId, payerId
onerror Occurs when a transaction is unable to be completed due to an error.
Note: cancel and complete will not fire.		 Error: message, [status], [code]
Methods
Execute(): boolean Initiates a purchase transaction based on the information set on the BuyNow object. Note that is an asynchronous method.
AddItem(PayPal.Checkout.Item item): void Adds a PayPal.Checkout.Item to the current purchase.
clearItems() Clears all items from the current purchase.
Properties
Name (Type) Description
merchantId (String) The merchant ID for the current transaction. This is the emailid of your PayPal Merchant account.
UseSandbox (bool) Set to true to use the PayPal Sandbox environment, false to use the live PayPal environment. Defaults to false.
displayShipping (Number) (optional, defaults to 0) Indicates how to display shipping options:
    0 = Show the shipping address.
    1 = Hide the shipping address.
    2 = Use the address from the buyers profile if one isn't provided.

Note: This property is automatically set to 1 for Digital Goods transactions.
confirmedShippingOnly (Boolean) (optional, defaults to false) Set to true if you require the buyer to have a confirmed shipping address.
Note: This property is automatically set to false for Digital Goods transactions.
currency (String) (optional, defaults to 'USD') Set to the three-character ISO-4217 currency code for this purchase. See table 2 in https://developer.paypal.com/webapps/developer/docs/classic/api/currency_codes/ for full list of supported currencies.
IPN (String) (optional) An Instant Payment Notification URL to which PayPal should POST purchase information.
custom (String) (optional) An arbitrary custom string value to associate with the current transaction. (NOTE: This field is limited to 256 single-byte alphanumeric characters.)
locale (String) (optional, defaults to 'en_US') Set to the five-character locale code to control the locale of PayPal pages.
paymentMethod (Paypal.Checkout.PaymentMethod) (optional, defaults to 'PayPal') Selects which payment flow is rendered by PayPal - the standard PayPal login page or the guest-checkout page for paying using a credit card.

PayPal.Checkout.Item object

This is an immutable object that describes the item being purchased. It can be created using PayPal.Checkout.ItemBuilder.

Properties
Name (Type) Description
name (String) The name of the item to be purchased. Should be limited to 127 single-byte characters.
description (String) (optional) The description of the item to be purchased. Should be limited to 127 single-byte characters.
id (String) (optional) The ID of the item to be purchased. Should be limited to 127 single-byte characters.
price (String) The price of the item to be purchased. Must be a positive number with 2 decimal places. The decimal separator must be a period (.), and the optional thousands separator must be a comma (,).
tax (String) (optional) The tax on the item to be purchased. . Must be a positive number with 2 decimal places. The decimal separator must be a period (.), and the optional thousands separator must be a comma (,).
quantity (Number) (optional, defaults to 1) Quantity of this item to be purchased. Must be a positive integer value.
metadata (Object) (optional) Arbitrary data used to describe the item. (e.g. setting "type" to "Digital" will mark this item as a Digital goods purchase.)

PayPal.Checkout.ItemBuilder object

This object is a builder used to create item objects.

Methods
Name(String name): ItemBuilder Sets the name of the item to be purchased.
Description(String description): ItemBuilder Sets the description of the item to be purchased.
Id(String id): ItemBuilder Sets the ID of the item to be purchased.
Price(String price): ItemBuilder Sets the price of the item to be purchased.
Tax(String tax): ItemBuilder Sets the tax on the item to be purchased.
Quantity(Number quantity): ItemBuilder Sets the quantity of this item to be purchased. Defaults to 1.
AddMetadata(String key, String value): ItemBuilder Adds arbitrary data relating to the item to be purchased.
Note: This mainly aids in normalizing APIs and only specific keys are used.
For example, setting "type" to "Digital" marks the item as a Digital Goods item.
Build(): Item Builds a PayPal.Checkout.Item object from the provided configuration options.
